POLÍTICA DE PRIVACIDAD

GK Tracker
Última actualización: 18 de mayo de 2026

GK Tracker ("la aplicación") es una herramienta de registro estadístico para porteros de fútbol desarrollada por Xilokawa Design ("nosotros").

1. RESPONSABLE DEL TRATAMIENTO

Xilokawa Design — España
Contacto: xilokawa@gmail.com

2. DATOS QUE RECOGEMOS

NINGUNO.

La aplicación no recoge, almacena ni transmite ningún dato personal a servidores externos. Todos los datos generados durante el uso (perfiles de portero, sesiones, eventos, estadísticas, notas) se almacenan EXCLUSIVAMENTE en la base de datos local del dispositivo del usuario.

3. PERMISOS

La aplicación NO solicita acceso a cámara, micrófono, ubicación, contactos, calendario, almacenamiento externo ni ningún otro permiso sensible del dispositivo.

4. EXPORTACIONES

Cuando el usuario utiliza voluntariamente las funciones "Exportar CSV", "Exportar JPG" o "Backup .zip", el archivo se genera localmente y se entrega al sistema operativo mediante el diálogo nativo "Compartir". El usuario decide a qué aplicación (Drive, OneDrive, correo, etc.) envía el archivo. GK Tracker no participa en esa transferencia ni tiene visibilidad sobre ella.

5. SERVICIOS DE TERCEROS

La aplicación no integra ningún SDK de analítica, publicidad, crash reporting ni redes sociales.

6. MENORES

La aplicación es apta para todos los públicos. Al no recoger datos, no procesa información de menores.
